# Скринер Облигаций на базе ISS MOEX

Через запросы к бесплатному API Мосбиржи (ISS MOEX) собираю все облигации на бирже и некоторые их метрики:
- дата начала торгов
- дата погашения
- размеры купонов
- расчитанная мосбиржей доходность
- объемы торгов
.. и ряд других

Метрики сохраняю в SQLLite базу (простая, файловая, локальная). Для дальнейшего анализа. В коде есть несколько примеров
анализа и построений отчетов с использованием Pandas. Однако можно использовать любые инструменты для работы с SQL базами. 

## Как использовать

1. Установить питон и зависимости
python -m pip install -r requirements.txt

2. В консоли (cmd) перейти в директорию установки и 
`python main.py` 
для получения списка доступных комманд 

## Описание создаваемых колонок

**id** - Уникальный идентификатор записи (первичный ключ)<br>
**is_traded** - Флаг торгуется ли облигация в настоящее время<br>
**secid** - Биржевой идентификатор ценной бумаги<br>
**shortname** - Краткое название облигации<br>
**price** - Цена в процентах от номинала<br>
**yieldsec** - Расчетная доходность по данным Мосбиржи (неточная)<br>
**calc_yield** - Доходность, рассчитанная вручную<br>
**month_percent** - Процент дохода в месяц (ручной расчет)<br>
**total_percent** - Общий процент дохода (ручной расчет)<br>
**volume** - Объем торгов на последней сессии в выбранном режиме (board)<br>
**remaining_coupons** - Количество оставшихся купонных выплат<br>
**days_to_buyback** - Количество дней до офферты (досрочного выкупа)<br>
**days_to_coupondate** - Количество дней до следующей купонной выплаты<br>
**days_since_prev_coupon** - Количество дней с предыдущей купонной выплаты<br>
**days_to_finish** - Количество дней до погашения облигации<br>
**couponvalue** - Абсолютное значение купона в денежных единицах<br>
**couponpercent** - Процентное значение купона<br>
**accruedint** - Накопленный купонный доход (НКД)<br>
**listlevel** - Уровень листинга (1 - высший, 3 - низший)<br>
**bondtype** - Тип облигации<br>
**buybackdate** - Дата офферты (досрочного выкупа)<br>
**matdate** - Дата погашения облигации<br>
**tradedate** - Дата последних торгов (если торгов не было - дата проверки)<br>
**couponfrequency** - Частота выплаты купонов (раз в год)<br>
**coupondate** - Дата следующей купонной выплаты<br>
**updated** - Дата и время последнего обновления данных<br>
**emitent_id** - Идентификатор эмитента<br>
**type** - Категория облигации (корпоративная, ОФЗ, муниципальная)<br>
**typename** - Название категории облигации<br>
**primary_boardid** - Основной режим торгов (board)<br>
**issuedate** - Дата начала торгов облигацией<br>
**initialfacevalue** - Первоначальная номинальная стоимость<br>
**faceunit** - Валюта номинала<br>
**issuesize** - Объем выпуска (количество облигаций)<br>
**facevalue** - Текущая номинальная стоимость<br>
**isqualifiedinvestors** - Доступна только для квалифицированных инвесторов<br>
**earlyrepayment** - Возможность досрочного выкупа<br>
**_month_percent** - Проценты в месяц до даты офферты или погашения<br>
**_total_percent** - Общие проценты до даты офферты или погашения
