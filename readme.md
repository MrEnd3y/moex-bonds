# Скринер Облигаций на базе ISS MOEX

Через запросы к бесплатному API Мосбиржи (ISS MOEX) собираю все облигации на бирже и некоторые их метрики:
- дата начала торгов
- дата погашения
- размеры купонов
- расчитанная мосбиржей доходность
- объемы торгов
.. и ряд других

Метрики сохраняю в SQLLite базу (простая, файловая, локальная). Для дальнейшего анализа. В коде есть несколько примеров
анализа и построений отчетов с использованием Pandas. Однако можно использовать любые инструменты для работы с SQL базами. 

## Как использовать

1. Установить питон и зависимости
python -m pip install -r requirements.txt

2. В консоли (cmd) перейти в директорию установки и 
`python main.py` 
для получения списка доступных комманд 

## Описание создаваемых колонок

**id** - Уникальный идентификатор записи (первичный ключ)

**is_traded** - Флаг торгуется ли облигация в настоящее время

**secid** - Биржевой идентификатор ценной бумаги

**shortname** - Краткое название облигации

**price** - Цена в процентах от номинала

**yieldsec** - Расчетная доходность по данным Мосбиржи (неточная)

**calc_yield** - Доходность, рассчитанная вручную

**month_percent** - Процент дохода в месяц (ручной расчет)

**total_percent** - Общий процент дохода (ручной расчет)

**volume** - Объем торгов на последней сессии в выбранном режиме (board)

**remaining_coupons** - Количество оставшихся купонных выплат

**days_to_buyback** - Количество дней до офферты (досрочного выкупа)

**days_to_coupondate** - Количество дней до следующей купонной выплаты

**days_since_prev_coupon** - Количество дней с предыдущей купонной выплаты

**days_to_finish** - Количество дней до погашения облигации

**couponvalue** - Абсолютное значение купона в денежных единицах

**couponpercent** - Процентное значение купона

**accruedint** - Накопленный купонный доход (НКД)

**listlevel** - Уровень листинга (1 - высший, 3 - низший)

**bondtype** - Тип облигации

**buybackdate** - Дата офферты (досрочного выкупа)

**matdate** - Дата погашения облигации

**tradedate** - Дата последних торгов (если торгов не было - дата проверки)

**couponfrequency** - Частота выплаты купонов (раз в год)

**coupondate** - Дата следующей купонной выплаты

**updated** - Дата и время последнего обновления данных

**emitent_id** - Идентификатор эмитента

**type** - Категория облигации (корпоративная, ОФЗ, муниципальная)

**typename** - Название категории облигации

**primary_boardid** - Основной режим торгов (board)

**issuedate** - Дата начала торгов облигацией

**initialfacevalue** - Первоначальная номинальная стоимость

**faceunit** - Валюта номинала

**issuesize** - Объем выпуска (количество облигаций)

**facevalue** - Текущая номинальная стоимость

**isqualifiedinvestors** - Доступна только для квалифицированных инвесторов

**earlyrepayment** - Возможность досрочного выкупа

**_month_percent** - Проценты в месяц до даты офферты или погашения

**_total_percent** - Общие проценты до даты офферты или погашения
