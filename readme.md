# Скринер Облигаций на базе ISS MOEX

Через запросы к бесплатному API Мосбиржи (ISS MOEX) собираю все облигации на бирже и некоторые их метрики:
- дата начала торгов
- дата погашения
- размеры купонов
- расчитанная мосбиржей доходность
- объемы торгов
.. и ряд других

Метрики сохраняю в SQLLite базу (простая, файловая, локальная). Для дальнейшего анализа. В коде есть несколько примеров
анализа и построений отчетов с использованием Pandas. Однако можно использовать любые инструменты для работы с SQL базами. 

## Как использовать

1. Установить питон и зависимости
python -m pip install -r requirements.txt

2. В консоли (cmd) перейти в директорию установки и 
`python main.py` 
для получения списка доступных комманд 

## Описание создаваемых колонок

| Поле | Тип данных | Описание |
|------|------------|----------|
| id | Integer | Уникальный идентификатор записи (первичный ключ) |
| is_traded | Boolean | Флаг торгуется ли облигация в настоящее время |
| secid | String | Биржевой идентификатор ценной бумаги |
| shortname | String | Краткое название облигации |
| price | Float | Цена в процентах от номинала |
| yieldsec | Float | Расчетная доходность по данным Мосбиржи (неточная) |
| calc_yield | Float | Доходность, рассчитанная вручную |
| month_percent | Float | Процент дохода в месяц (ручной расчет) |
| total_percent | Float | Общий процент дохода (ручной расчет) |
| volume | Integer | Объем торгов на последней сессии в выбранном режиме (board) |
| remaining_coupons | Integer | Количество оставшихся купонных выплат |
| days_to_buyback | Float | Количество дней до офферты (досрочного выкупа) |
| days_to_coupondate | Float | Количество дней до следующей купонной выплаты |
| days_since_prev_coupon | Float | Количество дней с предыдущей купонной выплаты |
| days_to_finish | Float | Количество дней до погашения облигации |
| couponvalue | Float | Абсолютное значение купона в денежных единицах |
| couponpercent | Float | Процентное значение купона |
| accruedint | Float | Накопленный купонный доход (НКД) |
| listlevel | Integer | Уровень листинга (1 - высший, 3 - низший) |
| bondtype | String | Тип облигации |
| buybackdate | DateTime | Дата офферты (досрочного выкупа) |
| matdate | DateTime | Дата погашения облигации |
| tradedate | DateTime | Дата последних торгов (если торгов не было - дата проверки) |
| couponfrequency | Integer | Частота выплаты купонов (раз в год) |
| coupondate | DateTime | Дата следующей купонной выплаты |
| updated | DateTime | Дата и время последнего обновления данных |
| emitent_id | Integer | Идентификатор эмитента |
| type | String | Категория облигации (корпоративная, ОФЗ, муниципальная) |
| typename | String | Название категории облигации |
| primary_boardid | String | Основной режим торгов (board) |
| issuedate | DateTime | Дата начала торгов облигацией |
| initialfacevalue | Float | Первоначальная номинальная стоимость |
| faceunit | String | Валюта номинала |
| issuesize | Integer | Объем выпуска (количество облигаций) |
| facevalue | Float | Текущая номинальная стоимость |
| isqualifiedinvestors | Boolean | Доступна только для квалифицированных инвесторов |
| earlyrepayment | Boolean | Возможность досрочного выкупа |
| _month_percent | Float | Проценты в месяц до даты офферты или погашения |
| _total_percent | Float | Общие проценты до даты офферты или погашения |
